Sejarah CSS

CSS dibuat pada tahun 1994 oleh Håkon Wium Lie. Håkon Wium Lie adalah seorang ahli komputer asal Norwegia yang bekerja di CERN. CSS dibuat untuk memisahkan antara konten dengan tampilan. Sebelum CSS ditemukan, tampilan dari halaman web dituliskan di dalam tag HTML. Hal ini menyebabkan ketika kita ingin mengubah tampilan, kita harus mengubah tag HTML-nya. Dengan adanya CSS, kita dapat mengubah tampilan halaman web tanpa harus mengubah tag HTML-nya.
